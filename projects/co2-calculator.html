CO₂ Freight Emissions Calculator

Python tool using GLEC Framework emission factors to calculate freight carbon footprint by route and mode, with lowest-carbon alternative suggestions.

🐍 Deliverable: Python CLI script + README
📐 Standard: GLEC Framework v3
🚛 Modes: Road, Sea, Air, Rail

The Problem

Since 2024, large European companies must report Scope 3 emissions under the Corporate Sustainability Reporting Directive (CSRD). Freight transport is typically the largest component of a manufacturer's or retailer's Scope 3 footprint. Yet most companies — especially mid-size ones — still rely on rough estimates or nothing at all.

This tool gives any logistics team a quick, standards-based way to calculate the CO₂ footprint of a specific shipment and immediately see whether a different mode would reduce it.

The Standard: GLEC Framework

The Global Logistics Emissions Council (GLEC) Framework is the industry standard for freight emissions calculation, recognized by the Science Based Targets initiative (SBTi) and used by major logistics providers (DHL, Maersk, DB Schenker). It defines emission intensity factors in kg CO₂e per tonne-km for each transport mode.

Why this matters for recruiters: CSRD awareness and emissions calculation capability are increasingly listed as requirements in supply chain analyst and procurement job descriptions in Europe. This project shows you understand the regulatory context, not just the math.

0.008

0.601

0.028

How It Works

# emission_factors.py — GLEC Framework v3 FACTORS = { "road": 0.062, # kg CO2e / tonne-km (full truckload) "sea": 0.008, # kg CO2e / tonne-km (container, avg) "air": 0.601, # kg CO2e / tonne-km (belly + freighter avg) "rail": 0.028, # kg CO2e / tonne-km (European avg) } def calculate_emissions(weight_t, distance_km, mode): """Returns CO2e in kg for a given shipment.""" factor = FACTORS[mode] return weight_t * distance_km * factor def best_alternative(weight_t, distance_km, current_mode): """Returns the lowest-carbon viable mode.""" results = { m: calculate_emissions(weight_t, distance_km, m) for m in FACTORS if m != current_mode } return min(results, key=results.get), min(results.values())

Limitations & Assumptions

  • Uses average emission factors — actual values vary by vessel age, load factor, fuel type
  • Does not account for refrigerated transport (higher energy intensity)
  • Distance is straight-line or user-provided — real routing adds 10–30% for road
  • Future version: integrate a distance API and per-carrier emission profiles
