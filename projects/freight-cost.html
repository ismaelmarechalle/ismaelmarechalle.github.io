Operational Case Excel

Freight Cost Comparison Tool

Interactive Excel workbook comparing road, rail, and sea freight across 6 routes — returning cost, transit time, CO₂, and an automated best-option recommendation.

📊 Deliverable: Interactive Excel workbook
🗺️ Routes: 6 origin/destination pairs
🚛 Modes: Road, Rail, Sea

The Problem

Every time a shipper needs to move cargo between two points, they face the same three-way trade-off: cost, speed, and carbon footprint. In practice, most companies make this decision informally — a quick call to a freight forwarder, a rule-of-thumb, or simple habit. A structured comparison tool makes the decision explicit, repeatable, and defensible.

This workbook lets a logistics coordinator or procurement analyst select a route and cargo weight and instantly see a side-by-side comparison across all three modes, with a recommendation logic built in.

The 6 Routes

Route Modes Available Context
Dakar → Abidjan Road, Sea West Africa coastal corridor
Marseille → Lyon Road, Rail France domestic, rail-competitive
Barcelona → Paris Road, Rail, Sea Iberian corridor, all modes viable
Hamburg → Warsaw Road, Rail Central Europe, growing rail network
Shanghai → Le Havre Sea, Air (for comparison) Long-haul intercontinental
Casablanca → Dakar Road, Sea North–West Africa, mixed infrastructure

Workbook Structure

  1. Input Sheet — user selects route (dropdown) and enters cargo weight in tonnes. All other calculations update automatically via named range references.
  2. Rate Matrix (hidden) — cost per tonne by route and mode (€), transit time in days, CO₂ factor in kg per tonne. This sheet is the single source of truth — updating rates here cascades everywhere.
  3. Comparison Output — side-by-side table showing total cost (€), transit time (days), CO₂ (kg), and a cost-per-day-in-transit metric that captures the speed-cost trade-off in one number.
  4. Recommendation Logic — three IF/INDEX formulas that output: "Best Cost," "Best Time," and "Best Carbon" mode. A fourth cell outputs a weighted recommendation based on user-defined priority weights (sliders via data validation).
  5. Chart Tab — clustered bar chart auto-updating from the comparison output, ready to paste into a slide or report.

Key Excel Techniques

  • Named ranges for all rate inputs — no hardcoded numbers in formulas
  • Data validation dropdowns for route and priority selection
  • INDEX/MATCH for dynamic rate lookup across route × mode matrix
  • Weighted scoring formula: = cost_weight * norm_cost + time_weight * norm_time + carbon_weight * norm_carbon
  • Conditional formatting to highlight the winning mode per criterion
  • Dynamic chart title using cell reference to show selected route name
